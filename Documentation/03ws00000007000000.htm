Install a localized ArcGIS Pro setup

A localized setup (Setup.exe) includes setup files for both the core ArcGIS Pro product and a specific language pack:

  • ArcGISPro.msi
  • Pro<language>LP.msi

Both setups are installed by Setup.exe. For example, if you run Setup.exe for ArcGIS Pro Spanish, both ArcGIS Pro and the Spanish language pack are installed.

Two different Add/Remove Program items are created for ArcGIS Pro and the language pack. Uninstalling one does not uninstall the other.

Additional language packs are installed separately. To install a second language pack, you run only the Pro<language>LP.msi file for the language.

ArcGIS Pro supports the following languages:

  • Arabic
  • Chinese (Simplified)
  • Chinese (Traditional)
  • Czech
  • Finnish
  • French
  • German
  • Hebrew
  • Italian
  • Japanese
  • Korean
  • Polish
  • Portuguese (Brazilian)
  • Russian
  • Spanish
  • Swedish
  • Turkish
  • Ukrainian

Note:

When running a localized ArcGIS Pro installation, the option to run ArcGIS Pro when the setup completes appears on the Ready to Install dialog box. If the option is left selected, ArcGIS Pro starts after the installation of the core product and its corresponding language pack. ArcGIS Pro does not start automatically after a silent installation.
